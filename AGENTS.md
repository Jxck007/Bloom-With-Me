# Bloom With Me repository rules

- Inspect before editing.
- Preserve working functionality.
- Use `public/assets/asset-map.json` as the artwork source of truth.
- Never replace supplied artwork with CSS flowers or emojis.
- Hand and voice are primary controls.
- Touch is always available as a friendly fallback.
- Never request camera or microphone automatically; require an explicit user action.
- Do not add a timer, lives, score penalty, game-over screen, or flashing effects.
- Keep sound muted by default.
- Run `npm run build` after every major phase.
- Test at 360x800, 768x1024, and 1366x768.
- Do not claim a feature works without testing it.
